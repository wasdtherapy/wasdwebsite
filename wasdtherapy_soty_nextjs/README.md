# wasd/therapy

Пространство спокойствия: дыхание, генеративные звуки, фокус, медитация и аффирмации. Без рекламы, регистрации и трекеров.

## Запуск локально

```bash
npm install
npm run dev
```

Открой http://localhost:3000

## Деплой на Vercel

1. Залей содержимое этой папки в GitHub-репозиторий (в корень).
2. На Vercel: Add New → Project → Import репозиторий.
3. Framework Preset: **Next.js** (определяется автоматически).
4. Deploy.

## Стек

- Next.js 14 (App Router)
- React Three Fiber + GLSL (живой орб, реагирующий на звук)
- Tone.js (генеративный эмбиент и микшер из 9 слоёв)
- GSAP (курсор, магнитные кнопки, переходы)
- Zustand (состояние, сохранение в localStorage)
- Tailwind CSS

## Разделы

- **/** — главная с живым орбом
- **/breathe** — 6 дыхательных техник
- **/sounds** — генеративный микшер атмосфер
- **/focus** — Pomodoro с эмбиентом
- **/meditate** — гайдовые медитации 3/5/10 мин
- **/affirmations** — аффирмации с избранным
- **/about** — манифест
